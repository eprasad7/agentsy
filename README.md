# Agentsy

Agentsy is an agent OS for companies that want to build, run, and govern agents without owning all the infrastructure themselves.

This repo is starting greenfield. The first artifact is an architecture and research blueprint that maps what an enterprise-grade agent stack needs as of March 19, 2026.
